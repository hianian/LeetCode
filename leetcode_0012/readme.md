## qurestion: 二叉树的后序遍历（简单）

给你一棵二叉树的根节点 root ，返回其节点值的 后序遍历 。

来源：力扣（LeetCode）
链接：https://leetcode-cn.com/problems/binary-tree-postorder-traversal 

示例 1：

![示例1](https://assets.leetcode.com/uploads/2020/08/28/pre1.jpg)

```text
输入：root = [1,null,2,3]
输出：[3,2,1]
```

示例 2：
```text
输入：root = []
输出：[]
```

示例 3：
```text
输入：root = [1]
输出：[1]
```

提示：
```text
树中节点的数目在范围 [0, 100] 内
-100 <= Node.val <= 100
```
